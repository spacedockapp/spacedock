Space Dock

About

Space Dock is an application for Macintosh, iOS and Android for putting together squadrons for WizK!d's excellent Star Trek: Attack Wing game.

Modest efforts are made to make sure Space Dock is working correctly when released but glitches happen. Make sure to export your most valuable squad designs for safe keeping.

Space Dock Mac requires a Mac running OS X 10.8 or later due to a dependency on application development features available in that version or later.

Space Dock iOS requires iOS version 6 or later.

Space Dock Android requires Android version 4.0 or later.

Questions about Space Dock? Comment? Email me at rob@tsuk.com.

Testing

If you'd like to help test versions of Space Dock for iOS before they are released to the store please visit this page at TestFlight. Since test devices are limited to 100 please only sign up if you will actually install and use test versions.

If you'd like to help test versions of Space Dock for Android before they are released to the store please join testing Google+ community and ask to join. Once joined you can download the alpha test version from the Google Play store.

Test versions for the Mac are available on this page.

Development

Space Dock is open source software. If you're interested in perusing the source code or helping out with development you can visit the repository on GitHub.
